[中文介绍](#中文介绍) | [English](#english)

## 中文介绍

`壹零捌学AI` 是一个面向 AI 自学与实践的资源仓库，用来沉淀我在学习和使用 AI 过程中遇到的有价值项目、工具、技能、平台和学习路线。

这个仓库不是通用的“AI 资源大全”，而是更偏向：

- 自学过程中实际会反复用到的工具与平台
- 与 Claude Code、Codex、Cursor、OpenClaw 等 agent 工作流相关的技能与项目
- 值得跟踪的 builder、开源项目和学习型仓库
- 与 AI 设计、CLI、自动化、学习平台、多智能体相关的入口整理

## English

`108 Learn AI` is a curated repository for AI self-learning resources, tools, skills, agent ecosystems, and project-based learning paths.

It focuses less on being a generic “awesome list” and more on collecting practical, reusable references that are actually useful during day-to-day AI learning and building.

## What this repository covers

- AI tools and platforms
- Skill libraries and agent workflow resources
- Agent ecosystems such as OpenClaw and related projects
- Project-based learning routes
- Builder tracking and information sources
- AI-product / CLI / workflow entry points
